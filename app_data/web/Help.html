Course Site Generator is a desktop Java application for creating and editing a course webpage. The app presents five tabs, one for each of the websites pages. The Site tab contains general information regarding the course, instructor, and styling of the site. The Syllabus tab contains the information regarding the syllabus for the course. The Meeting Times tab contains the information regarding the meeting of the class lectures, labs, and recitations. The Office Hours Tab contains information regarding Teaching Assistants (TAs) and their appropriate office hours. The Schedule Tab contains the information regarding holidays, homeworks, lectures, and any other dates deemed important for the calendar portion of the site. The application saves data to a JSON format and exports toan HTML file.

Step 1: Edit Site Information

To start, edit the infomation of the site regarding core information and styling preferences

Step 2: Edit Syllabus

Input the appropriate information regarding each portion of the syllabus.

Step 3: Edit Meeting Times

Input the proper Meeting times you want reflected on the site.

Step 4: Edit Office Hours

To start, input the TAs names and email addresses. Then select a TA and select the appropriate time in the Office Hours table to toggle the TA.

Step 5: Edit Scheule

Add or Edit any remaining schedule items you would like to reflect on the site calendar.

Step 5: Save and Export

Save the appropriate file and export it to an HTML for internet use.
